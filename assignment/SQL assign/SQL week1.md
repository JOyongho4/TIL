1-2 BigQuery 환경 설정

BigQuery 환경 구성 요소
1. 프로젝트: 하나의 큰 건물. 건물 안에는 여러 목적을 가진 창고가 존재. 하나의 프로젝트, 여러 데이터셋
2. 데이터셋: 프로젝트에 있는 창고. 창고에 데이터 저장. 별도의 데이터 저장가능. 하나의 데이터셋에 다양한 테이블 존재
3. 테이블: 창고에 있는 선반. 테이블 안에 행과 열로 이루어진 데이터들 저장
```
